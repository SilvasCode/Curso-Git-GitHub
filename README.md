# 🛠️ Curso de Git e GitHub (Curso em Vídeo)

Este repositório contém as anotações, exercícios e o projeto prático desenvolvidos durante o curso de **Git e GitHub**, ministrado pelo professor Gustavo Guanabara no [Curso em Vídeo](https://www.cursoemvideo.com/).

## 🎯 Objetivo
O objetivo deste curso foi dominar o controle de versão de códigos, aprendendo a utilizar o Git localmente e o GitHub para hospedagem e colaboração.

## 📄 Conteúdo do Repositório
* `Ola-Mundo.html`: O primeiro arquivo HTML criado para testar os commits.
* `README.md`: Este guia de referência.

## 🚀 Principais Comandos Aprendidos

Abaixo estão os comandos fundamentais que pratiquei durante o curso:

| Comando | Descrição |
| :--- | :--- |
| `git init` | Inicializa um novo repositório Git local. |
| `git status` | Verifica o estado dos arquivos (working directory e staging area). |
| `git add <arquivo>` | Adiciona um arquivo específico à staging area. |
| `git add .` | Adiciona todos os arquivos modificados à staging area. |
| `git commit -m "mensagem"` | Salva as alterações da staging area com uma mensagem descritiva. |
| `git log` | Exibe o histórico de commits. |
| `git remote add origin <url>` | Conecta o repositório local a um repositório remoto (GitHub). |
| `git push -u origin <branch>` | Envia os commits locais para o repositório remoto pela primeira vez. |
| `git push` | Envia os commits locais subsequentes. |
| `git pull` | Baixa e mescla as alterações do repositório remoto no local. |
| `git clone <url>` | Clona um repositório remoto existente para o seu computador. |
| `git branch -M main` | Renomeia a branch principal para `main`. |
